# DBA

Role: Database Administrator (DBA)

Responsibilities:
- Set up and maintain local development database (recommend Postgres)
- Define and maintain DB schema; own migrations and versioning
- Implement backup, restore, and local seed data processes
- Collaborate with DevOps to plan cloud DB migration and ensure migration-safe schema
- Support developers with schema changes and performance guidance
